MIKA AGUILAR

I am a web designer and and front-end developer with full-stack aspirations working near Austin, TX. This June I moved from Michigan to Texas and am now taking on development full-time here. I'm incredibly excited for my future in this great city!

What I Do

I love spending time developing front-end web applications and websites so I'm constantly using or trying out the latest applicable web technologies. My designs are done in Photoshop or Illustrator and I use HTML5, CSS3, jQuery and Javascript magic to make fully responsive, great looking stuff. When I write code or use it in a project, I make it clean, reusable, and responsive.

Front End

Front end development is my bread and butter these days. I've been using HTML5, CSS3, and Javascript enough to be sure I can make almost anything happen client side, and usually in more than one way which is the cool part of working with the web. There are so many cool things happening in front end development across the board, and finding cool new ways to do things is really fun for me. Recently I've been using a lot of javascript, and finding it a very powerful tool.

Design

I've taken several classes dealing with Photoshop, Illustrator and good old Flash. With photoshop I'm very confident making some pretty cool, advanced things happen from seamless cropping to photorealistic effects. I've also used Illustrator quite a bit and I'm far from a beginner with the pen tool. On top of being great with the required tools, I've spent a lot of time reading up on modern design practices and feel I have a decent grasp of whats going on in the web design world.
